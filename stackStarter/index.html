Divs

Divs are one of the most important things within html

They allow us to structure our pages and play with specified elements

This div has an id attribute that we called "first"

"First" is an arbitrary value we could call anything we wanted

The ID attribute of "first" allows us to specify this specific div.

IT is best practice to only use an ID attribute on one element. Just like your name or social security number should only point to you.

It will apply to everyting between the div tags

This div has a class attribute that we called "box"

"Box" is an arbitrary value we could call anything we wanted

The class attribute allows us to specify ANY div that has the class "box".

It will apply to everyting between the div tags
